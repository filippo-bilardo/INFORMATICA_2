Esempio Pratico: Catalogo Prodotti

Questo esempio mostra come utilizzare gli oggetti per gestire un catalogo prodotti con diverse categorie.

// Esempio di oggetto complesso con metodi let catalogo = { elettronica: [], abbigliamento: [], alimentari: [], aggiungiProdotto: function(categoria, prodotto) { if (this[categoria]) { this[categoria].push(prodotto); return true; } return false; }, trovaPerPrezzo: function(prezzoMax) { let risultati = []; for (let cat in this) { if (Array.isArray(this[cat])) { risultati = risultati.concat( this[cat].filter(p => p.prezzo <= prezzoMax) ); } } return risultati; } };
