Dichiarazione e Invocazione di Funzioni

Dichiarazione di una funzione

Una funzione può essere dichiarata in JavaScript utilizzando la parola chiave function, seguita dal nome della funzione, un insieme di parametri racchiusi tra parentesi e il corpo della funzione racchiuso tra parentesi graffe.

function nomeFunzione(parametro1, parametro2) {
  // Corpo della funzione
}

Esempio: Funzione di saluto

function saluta(nome) {
  return "Ciao, " + nome + "!";
}

Parametri e argomenti

I parametri sono variabili definite nella dichiarazione di una funzione. Gli argomenti sono i valori specifici passati alla funzione quando viene invocata.

Esempio: Funzione con più parametri

function somma(a, b) {
  return a + b;
}

Valore di ritorno

Le funzioni possono restituire un valore usando la parola chiave return. Una volta eseguito il return, la funzione si interrompe.

Esempio: Calcolo del quadrato

function quadrato(num) {
  return num * num;
}

Funzioni senza valore di ritorno

Non tutte le funzioni devono restituire un valore. Se non viene specificato un return, la funzione restituisce implicitamente undefined.

Esempio: Stampa messaggio

function stampaMessaggio() {
  console.log("Questo è un messaggio.");
  // Nessun return esplicito
}

Funzioni pure e impure

Le funzioni pure restituiscono sempre lo stesso output per gli stessi input e non producono effetti collaterali. Le funzioni impure possono dipendere da variabili esterne o modificarle.

Esempio: Funzione pura

function sommaPura(a, b) {
  return a + b;
}

Esempio: Funzione impura

let contatore = 0;

function incrementaContatore() {
  contatore++;
  return contatore;
}
