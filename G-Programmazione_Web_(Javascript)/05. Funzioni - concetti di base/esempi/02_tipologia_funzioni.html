Tipologia di Funzioni

Funzioni dichiarate (function declaration)

Le funzioni dichiarate sono definite con la parola chiave function seguita dal nome. Possono essere invocate anche prima della loro dichiarazione grazie all'hoisting.

function saluta(nome) {
  return "Ciao, " + nome + "!";
}

Funzioni assegnate a variabili (function expressions)

Le function expressions sono funzioni assegnate a variabili. Non vengono sottoposte a hoisting, quindi devono essere definite prima dell'invocazione.

const moltiplica = function(a, b) {
  return a * b;
};

Funzioni anonime

Le funzioni anonime sono funzioni senza nome, spesso utilizzate come callback.

setTimeout(function() {
  console.log("Messaggio ritardato");
}, 1000);

Funzioni freccia (arrow functions)

Le arrow functions sono una sintassi compatta per dichiarare funzioni, introdotta in ES6.

const somma = (a, b) => a + b;

Funzioni di callback

Una funzione di callback è una funzione passata come argomento a un'altra funzione.

function esegui(funzione) {
  funzione();
}

Funzioni di ordine superiore

Le funzioni di ordine superiore sono funzioni che accettano altre funzioni come argomenti o restituiscono funzioni.

function applicaOperazione(arr, operazione) {
  return arr.map(operazione);
}

Funzioni autoinvocate (IIFE)

Le IIFE (Immediately Invoked Function Expressions) sono funzioni che vengono dichiarate e immediatamente invocate.

(function() {
  console.log("Questa funzione si esegue subito!");
})();
