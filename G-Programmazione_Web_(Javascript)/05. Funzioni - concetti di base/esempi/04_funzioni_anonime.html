Funzioni Anonime

Dichiarare una funzione anonima

Le funzioni anonime sono funzioni senza nome, spesso assegnate a variabili o passate come callback.

const somma = function(a, b) {
  return a + b;
};

Funzioni anonime come callback

Le funzioni anonime sono spesso utilizzate come callback, ad esempio nei metodi degli array.

const numeri = [1, 2, 3, 4, 5];
const doppi = numeri.map(function(numero) {
  return numero * 2;
});

Funzioni anonime e arrow functions

Le arrow functions offrono una sintassi più compatta per le funzioni anonime.

const numeri = [1, 2, 3, 4, 5];
const doppi = numeri.map(numero => numero * 2);

Funzioni autoinvocate anonime (IIFE)

Le IIFE sono funzioni anonime che vengono dichiarate e immediatamente invocate. Sono utili per creare un ambito privato.

(function() {
  console.log("Questa funzione si esegue subito!");
})();

Funzioni anonime in eventi

Le funzioni anonime sono spesso utilizzate come gestori di eventi.

document.getElementById("bottone").addEventListener("click", function() {
  console.log("Bottone cliccato!");
});
