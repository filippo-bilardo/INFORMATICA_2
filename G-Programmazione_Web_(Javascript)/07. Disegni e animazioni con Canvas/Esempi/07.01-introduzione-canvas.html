Introduzione a Canvas HTML5

L'elemento <canvas> è un componente HTML che permette di disegnare grafica e animazioni utilizzando JavaScript. È stato introdotto con HTML5 ed è ampiamente supportato dai browser moderni.

Il tuo primo Canvas

Ecco un canvas base con una dimensione di 400x200 pixel:

<canvas id="primoCanvas" width="400" height="200"></canvas>

Disegnare sul Canvas

Per disegnare sul canvas, è necessario ottenere il suo contesto di rendering. Il più comune è il contesto 2D, che consente di disegnare forme bidimensionali, testo, immagini e altro. 

// Ottieni il contesto 2D del canvas
const canvas = document.getElementById('secondoCanvas');
const ctx = canvas.getContext('2d');

// Disegna un rettangolo
function disegnaRettangolo() {
    ctx.fillStyle = 'blue';
    ctx.fillRect(50, 50, 150, 100);
}

// Disegna un cerchio
function disegnaCircolo() {
    ctx.beginPath();
    ctx.arc(300, 100, 70, 0, Math.PI * 2);
    ctx.fillStyle = 'red';
    ctx.fill();
}

Canvas responsive

Per rendere il canvas responsive, è possibile utilizzare CSS per impostare la larghezza al 100% e calcolare l'altezza proporzionalmente tramite JavaScript.

Nota importante: Quando si ridimensiona un canvas con CSS, è necessario tenere conto del rapporto tra le dimensioni in pixel del canvas (width e height) e le dimensioni di visualizzazione.

Considerazioni sulla compatibilità

Sebbene il supporto per Canvas sia eccellente nei browser moderni, è sempre buona pratica fornire contenuto alternativo per browser più vecchi o situazioni in cui JavaScript è disabilitato. 

<canvas id="myCanvas" width="400" height="200">
    Il tuo browser non supporta l'elemento Canvas.
    Ecco un'immagine statica al suo posto.
    <img src="fallback-image.png" alt="Alternativa al canvas">
</canvas>
