Disegno di Forme Semplici in Canvas

1. Rettangoli

I rettangoli sono le forme pi√π semplici da disegnare in Canvas.

// Rettangolo pieno
ctx.fillStyle = 'blue';
ctx.fillRect(x, y, width, height);

// Rettangolo con solo bordo
ctx.strokeStyle = 'red';
ctx.lineWidth = 2;
ctx.strokeRect(x, y, width, height);

// Cancella un'area rettangolare
ctx.clearRect(x, y, width, height);

2. Linee

Le linee si disegnano definendo un percorso (path) e poi tracciandolo.

// Inizio un nuovo percorso
ctx.beginPath();

// Sposto il "pennino" in posizione iniziale
ctx.moveTo(x1, y1);

// Traccio la linea fino al punto finale
ctx.lineTo(x2, y2);

// Imposto lo stile della linea
ctx.strokeStyle = 'black';
ctx.lineWidth = 2;

// Disegno effettivamente la linea
ctx.stroke();

3. Cerchi e Archi

Cerchi e archi si creano con il metodo arc().

// Inizio un nuovo percorso
ctx.beginPath();

// Disegno un cerchio
// arc(x, y, raggio, angolo iniziale, angolo finale, antiorario)
ctx.arc(x, y, radius, 0, Math.PI * 2);

// Riempio il cerchio
ctx.fillStyle = 'orange';
ctx.fill();

// Oppure disegno solo il bordo
ctx.strokeStyle = 'blue';
ctx.lineWidth = 3;
ctx.stroke();

4. Poligoni

I poligoni vengono creati definendo i loro vertici come un percorso.

// Inizio un nuovo percorso
ctx.beginPath();

// Definisco i vertici
ctx.moveTo(x1, y1); // Primo vertice
ctx.lineTo(x2, y2); // Secondo vertice
ctx.lineTo(x3, y3); // Terzo vertice
// ...altri vertici...

// Chiudo il percorso tornando al punto iniziale
ctx.closePath();

// Riempio il poligono
ctx.fillStyle = 'purple';
ctx.fill();

// Opzionalmente disegno anche il bordo
ctx.strokeStyle = 'black';
ctx.lineWidth = 1;
ctx.stroke();
