# Corso C++ By Example

Benvenuto al corso "C++ By Example"! Questo corso è progettato per guidarti attraverso le caratteristiche fondamentali e avanzate del linguaggio C++, partendo dalle basi fino ad arrivare a concetti più complessi. Il corso è strutturato in 30 esercitazioni pratiche, ognuna accompagnata da guide teoriche dettagliate.

## Struttura del Corso

Ogni esercitazione si trova in una cartella dedicata con un prefisso numerico e un nome descrittivo (es. `01_hello_world_io`) e contiene:

*   Un file `readme.md` con la descrizione dell'esercizio e i link agli argomenti teorici correlati.
*   Una sottocartella `teoria` contenente le guide approfondite per ciascun argomento.
*   Il codice sorgente dell'esercizio (o le istruzioni per crearlo).
